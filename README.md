### 1. Ruby 

* [(link to Repl.it)](https://repl.it/@Jabernathy88/FizzBuzz-Ruby)

```
numbers = (1..101)

def fizz_buzz(input) 
  input.each do |num|
    if num % 15 == 0 
      puts "FizzBuzz"
    elsif num % 5 == 0 
      puts "Buzz"
    elsif num % 3 == 0 
      puts "Fizz"
    else 
      puts num
    end 
  end 
end 

fizz_buzz(numbers)
```
